 == Description ==

Электронные накладные Новой почты. Вывод номера накладной в заказе (woocommerce)
Для вывода Электронной накладной не только в таблице заказа используйте специальный шорткод. Например вывод накладной для заказа №10: [getnp_number order_id="10"].  Добавлен виджет новой почты для отслеживания накладной в заказе (woocommerce).

== Installation ==

1. Загрузить плагин в `/wp-content/plugins` папку
2. Активировать плагин в пункте меню 'Плагины' у WordPress

== Screenshots ==

2. Поле для изменения названия поля Электронной накладной
3. Добавлен виджет новой почты для отслеживания накладной
